Loon

Using Exemplars to Visualize Large Scale Microscopy Data

Publication Code

Loon is a visualization analysis tool for quantitative phase microscopy data. Due to the high-throughput of automated imaging systems, Loon selects exemplar data points to display for the user. Also, it supports multiple visualization windows that are linked together through selection and filtering.
